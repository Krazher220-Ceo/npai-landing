# NPAI - National Platform Analytic Industrial with AI

Маркетплейс кодов для IoT устройств, AI-моделей и решений для анализа данных подключенных систем.

## О проекте

NPAI — это онлайн-магазин готовых программ для умных устройств на заводах и в компаниях. Вместо месяцев разработки вы получаете решение за минуты.

## Структура проекта

- `landing-site/` - Рекламный сайт (GitHub Pages)
- `Program(site)/` - Основное приложение
- `functional-site/` - Функциональный сайт
- `backend/` - Backend API (в разработке)

## Быстрый старт

### Локальный запуск

```bash
cd landing-site
python3 -m http.server 8000
```

Откройте браузер: http://localhost:8000
