# NPAI Landing Site

Рекламный сайт для платформы NPAI.

## GitHub Pages

Этот сайт настроен для автоматической публикации на GitHub Pages.

### Настройка GitHub Pages

1. Перейдите в Settings репозитория
2. В разделе Pages выберите Source: `main` branch
3. Выберите папку: `/landing-site`
4. Сохраните изменения

Сайт будет доступен по адресу: `https://[username].github.io/[repository-name]/`

## Локальный запуск

```bash
python3 -m http.server 8000
```

Откройте браузер: http://localhost:8000

## Структура

- `index.html` - Главная страница
- `gallery.html` - Интерактивная галерея
- `styles/` - CSS стили
- `scripts/` - JavaScript файлы
- `*.png` - Изображения
